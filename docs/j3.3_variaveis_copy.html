Variáveis ‘factor’

  • Factorizar uma variável não factor
v <- c(0, 1, 1, 0, 5,5, 0)

f <- factor (v)
class(f)
## [1] "factor"
  • Visualizar ‘levels’ de uma factor
levels(f) 
## [1] "0" "1" "5"
  • Atribuir ‘labels’ de uma factor
f <- factor(f,
            levels = c(0,1, 5),
             labels = c("azul", "verde", "cinzento"))

f
## [1] azul     verde    verde    azul     cinzento cinzento azul    
## Levels: azul verde cinzento
  • Recodificar em 2 categorias
f.2 <- ifelse(f=="azul",  "A", "nonA") 
f.2
## [1] "A"    "nonA" "nonA" "A"    "nonA" "nonA" "A"


  • Recodificar em 3 categorias
age <- c(9, 9, 11, 14, 16, 30, 50)

age.3 <- NA
age.3  [age < 10]             <- "child"
age.3  [age >= 10 & age < 20] <- "teenager"
age.3  [age >= 20]            <- "adult"

age.3
## [1] "child"    "child"    "teenager" "teenager" "teenager" "adult"    "adult"

  • Dica para ordenar levels de uma factor baseado na ordem das linhas numa base aqui
    In short, if you have a data.frame with the rows in order and you want the levels of one of the columns to be in that same order,
    you can use ´Df <- mutate(Df, col = factor(col, col))`.
library(dplyr)
## 
## Attaching package: 'dplyr'
## The following objects are masked from 'package:plyr':
## 
##     arrange, count, desc, failwith, id, mutate, rename, summarise,
##     summarize
## The following objects are masked from 'package:stats':
## 
##     filter, lag
## The following objects are masked from 'package:base':
## 
##     intersect, setdiff, setequal, union
Df <- data.frame(
  num = 5:1,
  word = c("five", "four", "three", "two", "one"))
levels(Df$word) # levels are alphabetical by default
## NULL
Df$word # right now the order is 5,4,3,2,1
## [1] "five"  "four"  "three" "two"   "one"
Df <- Df %>%
  arrange(num) %>%   # rearrange the Df in the order we want (1,2,3,4,5)
  mutate(word = factor(word, word)) # this line reorders the factor in the same order
levels(Df$word)  # now the levels are 1,2,3,4,5
## [1] "one"   "two"   "three" "four"  "five"

NAs (valores em falta)

  • Detectar NAs numa variável
table (is.na(x))
  • Tabela que não exclua NAs
table (x, exclude = FALSE)


  • Calcular média duma variável com NAs
mean(x, na.rm = TRUE)


  • Eliminar observações com NA numa variável
Df <-subset( Df   ,   !is.na (sexo))


  • Eliminar obs com NA em todas as variables da base Df
Df <- na.omit (Df)


  • Recodificar NAs em 0
y[is.na(y)] <- 0         # transforma “NA” de “y” en “0”


  • Recodificar como NA
(D$x[D$x==88] <- NA # recodificar x==88  em NA


  • Listar linhas com NAs
Df[!complete.cases(Df),]


  • Criar nova base sem NAs
DF_new <- na.omit(DF)
