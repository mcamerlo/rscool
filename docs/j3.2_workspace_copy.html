Working Directory

Gestão do Workspace (ApêndiceWeb)

Momentos básicos de uma sessão R



O ficheiro .Rproj

Preparação do ambiente de trabalho com a criação de um ficheiro .Rproj

  1. Antes de iniciar R.Studio: criar a pasta do projeto
  1. Iniciar sessão R.Studio (clique no ícono) e digitar:
    New Project -> Existing Directory -> NomeDaPasta -> Create Project


Iniciar sessão

Clicar no ficheiro .Rproj


  • Obter informação sobre a sessaõ R
sessionInfo()
  • Opção numero de digitos para a sessão
options(digits=4)


Working directory (wd)

  • Localizar o wd
getwd()
  • Mudar localização wd
setwd ("/Users/Q/My_R/R_Project1")

R utiliza / (forward slashes) para designar subdirectorios (diferente ao uso no Windows, ugual a Mac e Unix)

  • Listar os ficheiros do wd
list.files()


A memoria ‘ram’ do Workspace

  • Listar objectos ativos nos workspace
ls ( )          # listar os objetos da sessão
ls(pat = "c")   # listar objetos que contêm ‘c’ nos seus nomes
ls(pat= "^c")   #  listar objects que comecem com  ‘c’
ls(pos=2)       # Check the contents of the file that has been attached
  • Remover objectos ativar
rm("var1" )         # remover objeto "var1"

rm(list =ls() ) # remover todos os objetos da sessão

rm(list=setdiff(ls(), "B"))   # remover todo menos "B"

rm(list=setdiff(ls(), c("W", "X")))   # remover todo menos "W" e "X"


Pacotes

  • Instalar pacotes externos
install.packages ("PackageName")
  • Activar pacotes já instalados
library (PackageName)       
                # Or: clicking packages on Window4/ Tab Packages
                # Or via menu system: Packages>Load Packages
  • Carregar bases internas
data(aids)    # carregar base ‘aids’ (pacote ‘(KMsurv)’
  • Fechar um pacote
detach ("package:name") #detach data
  • Mais sobre pacotes
search ()               # mostra pacotes abertos

library(help=KMsurv)        # mostra info sobre pacote KMsurv

.libPaths()                 # indica a localização da 'library', the ‘library’ subdirectory of R_HOME.

library ()          # cria script com todos os pacotes instalados e na library (ou olhar pestana FILES)

Carregar

  • Carregar objetos do working directory
load ("objecto_y")          
load ("objectox_yx")        
load (“Minhabase")
load ("WorkspaceProj1")
  • Carregar objetos de outros diretorios
load ("~/0R/AA")
  • Recolher a posição de um ficheiro
file.choose()


Salvar

  • Salvar objetos no working directory
save (y, file="object_y")               # Guardar y como "object_y"     
save (y, x, file="object_yx")           # Guardar y and x 
save (minhabase, file="Minhabase.RData")        # Guardar dataframe
save.image(file="WorkspaceProj1")       # Guardar workspace
  • Salvar objetos em outros directorios
save (AA, file= "~/0R/AA")  
        # AA= obeto R
        # "~/0R/AA"= posição e nome do objeto salvado


  • Salvar Gráficos para pdf
pdf("mygraph.pdf")      # create the file
plot (x,)       # execute the graphic
dev.off()       # finish the process

# http://www.statmethods.net/interface/io.html
  • Salvara outputs para um ‘text file’
  1. Usando sink() e imprimendo
# a) Criar o ficheiro
sink("meu_documento.txt")  

#  b) Especificar um título interno
print("The mean of var1 x is…") 

# c) produzir os dados (o output será imprimido no .txt, não na CONSOLA)
    mean (x)
    plot (x)

# d) Finalizar o procedimento:
    sink()

b Usando sink() e cat()

# a) Criar o ficheiro    
sink("meu_documento.txt")

# b) Concatenar    
cat("A média de varX é...", mean(varX), "\n") 

# c) Finalizar o procedimento:
sink()

Importar bases

Exportar

  • Para xlsx
library (openxlsx)   
write.xlsx(Df, "Df.xlsx")
  • Para CSV
write.table(Df, file="Df.csv",sep=",",row.names=F)
  • Para SPSS
library(foreign)
write.foreign(mydata, "c:/mydata.txt", "c:/mydata.sps",   package="SPSS")
  • Para SPSS
library(foreign)
write.dta(Df, "Df.dta")
